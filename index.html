🔒 I tuoi dati non lasciano mai il tuo dispositivo

Tutto il processing avviene nel browser: analisi del file, decodifica MIME, visualizzazione HTML. Nessuna connessione di rete viene aperta durante l'utilizzo. Nessun cookie, nessun localStorage, nessun tracking. Il codice sorgente è un singolo file HTML verificabile riga per riga.

Come funziona

In tre passi, senza installare nulla

1

Esporta da Google

Vai su takeout.google.com, seleziona Gmail e scarica lo ZIP del tuo archivio.

arrow_forward
2

Apri il Visualizzatore

Clicca il pulsante qui sopra. La pagina funziona offline dopo il primo caricamento.

arrow_forward
3

Importa e naviga

Clicca "Importa Mail", seleziona il file ZIP o MBOX e consulta la posta.

💡 Archivi grandi (> 400 MB ZIP): estrai lo ZIP sul computer e importa direttamente il file .mbox contenuto in Takeout/Mail/. Il file .mbox viene letto in streaming — nessun limite di dimensione.

Dettagli tecnici sulla privacy

Verificabile riga per riga nel codice sorgente

AspettoComportamento
Trasmissione dati❌ Nessuna — zero connessioni di rete durante l'uso
Cookie / localStorage❌ Non utilizzati
Analytics / tracking❌ Assenti
Codice sorgente✅ Un singolo file HTML, aperto e verificabile
Dipendenze esterne✅ Solo JSZip + DOMPurify (CDN, cached dopo il primo avvio)
Uso offline completo✅ Sostituisci i tag <script> con copie locali per uso air-gapped

🔒 Your data never leaves your device

All processing happens in the browser: file parsing, MIME decoding, HTML rendering. No network connections are opened during use. No cookies, no localStorage, no tracking. The source code is a single auditable HTML file.

Key Features

Everything you need to browse a Gmail archive, with no privacy compromises

Absolute Privacy

Streaming loading in 32 MB chunks. No size limit, no crashes. Tested up to 3 GB.

Tree Labels
